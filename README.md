# HomeBoard Add-on Repository

Home Assistant add-on repository for [HomeBoard](./homeboard) — a lightweight household Kanban board.

## Add-ons

### HomeBoard

A self-hosted household task board with three Kanban columns (To Do, In Progress, Done), drag-and-drop cards, due dates, notes, and configurable team member assignments. Includes read-only Home Assistant sensor endpoints for automations and dashboard widgets.

## Installation

1. In Home Assistant go to **Settings → Add-ons → Add-on Store**
2. Click the **⋮ menu** (top right) → **Repositories**
3. Paste your GitHub repo URL: `https://github.com/YOUR_GITHUB_USERNAME/homeboard-addon`
4. Click **Add** → **Close**
5. Find **HomeBoard** in the store and click **Install**
6. Enable **Show in sidebar** on the add-on page
7. Click **Start**
